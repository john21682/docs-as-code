# API Overview

The Sample API allows applications to manage users, orders, and payments.

## Base URL

```
https://api.samplecompany.com/v1
```

## Data Format

- All requests and responses use JSON
- UTF-8 encoding

## HTTP Methods

| Method | Usage |
|--------|--------|
| GET | Retrieve data |
| POST | Create data |
| PUT | Update data |
| DELETE | Delete data |
